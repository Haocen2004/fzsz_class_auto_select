# 福三中自动抢课助手
解放双手，自动抢课，福三中限定

# 使用方法
打开选课界面登录  
按下 F12 打开控制台  
切换到 `Console` 或 `控制台` 标签页  
复制 `main.js` 文件内容并修改目标时间戳  
全部粘贴并按下回车加载  
系统将会自动检测并应用对应版本
