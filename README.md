- 上网账号加密代码 `do_encrypt_rc4.js` 
- 宪法学习自动完成 `xianfa_auto.js`
- 网络评教辅助点击 `pingjiao_auto.js`

# 使用方法
打开对应界面，如果需要登录请先登录完毕  
按下 F12 打开开发者窗口   
切换到 `Console` 或 `控制台` 标签页  
复制文件 **全部** 内容粘贴到 `Console` 或 `控制台` 标签页并按下回车加载
